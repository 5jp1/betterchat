# **App Name**: BetterChat

## Core Features:

- User Authentication: Secure user registration and login with username and password.
- Global Chat Rooms: Join and participate in various global chat rooms hosted with JSON file.
- Real-time Messaging: Send and receive messages in real-time using JSON as the primary data format, with the server pulling updates every second.
- JSON Database: Use a JSON file as a database to store messages and chats.
- Friend System: Add friends and send direct messages to them.
- Profile Customization: Change your profile information and settings.
- Voice Recording: Record and send audio voice messages.
- Image Sending: Send images, which are stored as image files within the application's data structure.

## Style Guidelines:

- Primary color: Black for a modern, sleek look with a subtle blue glow effect.
- Secondary color: Blue as a secondary color to complement the black background, applied to text and buttons.
- Modern sans-serif font for readability and a clean aesthetic, using blue for emphasis.
- Simple, clean icons for navigation and actions, using a consistent line style in blue.
- A clean and intuitive layout with a focus on readability and ease of use, with a grid structure similar to Discord for channel and user organization. The design will be responsive to ensure a seamless experience on all devices, mirroring modern chat applications.
- Subtle animations to provide feedback and enhance user engagement, such as message send confirmations and loading indicators, consistent with modern chat applications, using blue highlights.
